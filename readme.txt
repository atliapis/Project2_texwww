Για το δεύτερο project του μαθηματος χρειάστηκε να δημιουργήσω ένα βιογραφικό του εαυτού μου και να το διαμορφώσω με τρεις διφορετικούς τρόπους. Οι τρεις που υλοποίησα είναι οι formal, europass, fancy και εφαρμόζονται αποκλειστικά από 3 διαφορετικά css. Τοποθέτησα 3 radio button για την εναλλαγή μεταξύ αυτών και οι παράμετροι αλλάζουν μέσω javascript.

Το html συντίθεται από πίνακες και επιτυγχάνεται responsiveness σε μεγάλο βαθμό.

Για την europass διαμόρφωση χρησιμοποιήθηκε υλικό από το επίσημο site: http://europass.cedefop.europa.eu/en/home
